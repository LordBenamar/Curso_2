Proyecto PHP

Realizar un sistema de gestion de inventario (CRUD), este sistema debe permitir:
- Insertar productos
- Editar productos
- Eliminar productos mediante nombre y id (Solo si ambos campos coinciden)
- Visualizar productos en una tabla o cuadricula. (Puedes usar tablas de HTML o Grid)


Ademas debe permitir: 
- Iniciar sesion (El usuario debe estar en la tabla usuarios y ser registrado manualmente en la BD)



Estructuras de las tablas:
- Productos: ID (llave primaria y autoincremental), Nombre, Descripcion, Precio, Colores, Stock, Precio con Descuento, Etiqueta de creado

- Usuarios: ID (llave primaria y autoincremental), Nombre, Apellido, Email, Password (Con Hash), Etiqueta de registrado



Puntos fuertes:
- No debe permitir SQL Injection, y si lo detecta que redirija a una pagina de trol
- Debe ser Responsive
- Debe contener modularizacion
- Animaciones visuales
